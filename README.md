## Project Details

### Project Overview

This project involves designing a tapeout-ready Application-Specific Integrated Circuit (ASIC) microcontroller using the open-source SkyWater PDK, which integrates a 32-bit RISC-V RV32I single-cycle CPU, on-chip SRAM, and external interfaces like JTAG and SPI. The primary deliverable is a verified GDSII file ready for fabrication.

### Prototyping
- Universal Verification Methodology (UVM) Testbench
- FPGA Hardware Prototyping
